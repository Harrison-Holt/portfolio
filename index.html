Web Accessibility Focus

As a developer with personal experience of disability, I am passionate about building accessible and inclusive web applications. My commitment to web accessibility stems from both my studies and my desire to ensure that the web is usable by everyone. Tools and Certifications To ensure the highest standards of accessibility, I utilize a range of tools and have obtained relevant certifications:

Accessibility Testing Tools:
  • Pa11y: An automated accessibility testing tool that helps identify issues in web applications.
  • Lighthouse: An open-source tool by Google for auditing web page performance and accessibility.
  • ANDI (Accessible Name & Description Inspector): A tool to test websites and web-based applications for accessibility.
  • >WAVE (Web Accessibility Evaluation Tool): A tool that provides visual feedback about the accessibility of web content.
  • axe: A suite of accessibility testing tools for HTML, Android, and iOS applications.
Certifications:
  • DHS Trusted Tester Certification: A certification that demonstrates proficiency in applying the DHS Trusted Tester Conformance Process for accessibility testing
